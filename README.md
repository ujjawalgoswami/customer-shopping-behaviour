# customer-shopping-behaviour
Project Overview

This project analyzes customer shopping behaviour using Python, SQL, and Power BI to identify customer purchasing patterns, sales trends, product performance, and business insights.
The goal of this project is to help businesses make data-driven decisions by understanding customer preferences and improving sales strategies.

Technologies Used
Python
        Pandas
        NumPy
        Matplotlib
        Seaborn
SQL
        Data Cleaning
        Data Transformation
        Aggregations
        Joins
        Customer Analysis Queries
Power BI
        Interactive Dashboards
        KPI Cards
        Sales Trend Analysis
        Customer Segmentation
        Product Performance Visualization
